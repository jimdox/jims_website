My current project I am working on is a physics simulation engine with real-time rendering in Vulkan. If your hardware isn't well supported on Vulkan, there is a compatibility profile that uses OpenGL instead.
You can view the project here.

Jim's Physics Library (JPL) is a simple vector math library which, although does not have the low level optimizations required to be especially fast, was sufficient for past, less computationally intensive physics projects, and is quite simple.

ORBT was my first dive into graphics programming in OpenGL using the Lightweight Java Game Library (LWJGL). This was a great learning experience for me to get a grasp on the basics of modern OpenGL and rendering techniques.

Haskmat is basically the Haskell vector math library that no one asked for. It also includes data structures like linked-lists, queues, and some other things that also probably don't belong in Haskell. :P
